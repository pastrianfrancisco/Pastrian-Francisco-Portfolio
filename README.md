## 📂 How to View Projects

1. Browse the `projects/` folder and open any project subfolder.
2. Read the project's `README.md` for context, goals, and key insights.
3. View screenshots in the `screenshots/` folder for a quick preview.
4. Download the `.pbix` file (if available) to explore the report in Power BI Desktop.

> **Note:** Some `.pbix` files may use sample/anonymized data to protect privacy.
